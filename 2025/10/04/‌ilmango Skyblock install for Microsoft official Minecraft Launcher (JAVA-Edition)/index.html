‌ilmango Skyblock install for Microsoft official Minecraft Launcher (JAVA-Edition)

Author: Menyang Gong Date: October 4, 2025  16:00:00

For Single-Player

  1. Install Minecraft Launcher from Microsoft Store
  2. Download and Install Fabric
  3. Download Fabric-API
  4. Download Fabric-Carpet
  5. Download CarpetSkyAdditions-Reborn or CarpetSkyAdditions (Just for Minecraft edition before 1.20.6)
  6. Download Cloth-Config-API
  7. Place the Fabric-API, Fabric-Carpet, CarpetSkyAdditions and Cloth-Config-API mods in the <minecraft file directory>/mods/ folder
  8. Start Minecraft and select Create New World
  9. Enable the carpetskyadditions/skyblock data pack
  10. Click More World Options
  11. Select World Type: SkyBlock
  12. Create a world and Enjoy

For Multi-player

This mod only requires server-side configuration.

  1. Download and Create a Fabric server
  2. Download Fabric-API
  3. Download Fabric-Carpet
  4. Download CarpetSkyAdditions-Reborn or CarpetSkyAdditions (Just for Minecraft edition before 1.20.6)
  5. Download Cloth-Config-API
  6. Start the server to generate the server.properties configuration template file
  7. Shut down the server
  8. Delete the world folder generated by the server (this means deleting the default world that was initially generated)
  9. Place the Fabric-API, Fabric-Carpet, CarpetSkyAdditions and Cloth-Config-API mods in the <server folder>/mods/ folder.
  10. Open the server.properties configuration file.
  11. Change level-type=minecraft\:normal (world generation type: default) to level-type=carpetskyadditions\:skyblock (world generation type: skyblock).
  12. If you modify other generation options or specify a world seed, do so here as well.
  13. Start the server.
  14. Enable the module’s built-in data pack by executing the datapack enable "carpetskyadditions/skyblock" command in the server console or in-game with OP privileges.
  15. Enjoy
